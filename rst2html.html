Directives t.b.v. formatting:
.. startc:: x
start kolomweergave van x cellen breed (x = 12 of 16)
.. firstc:: x [class]
start eerste kolom met breedte van x cellen met evt. class
.. nextc:: x [class]
start volgende kolom met breedte van x cellen met evt. class
.. clearc::
einde regel na laatste kolom
.. spacer:: [x]
maak lege ruimte (met breedte van x cellen)
.. bottom:: x [None | filename] [link text]
maak navigatiefooter van x cellen breed met "index" en "volgende" link, "made by" tekst en mailto link. De tekst voor de "volgende" link is standaard tenzij anders aangegeven;
bij ontbreken van de filenaam wordt ook geen index link weergegeven;
bij "None" wordt de index link wel getoond
.. endc::
einde kolomweergave
.. refkey:: trefwoord [ : anchor]
definieer een verwijzing naar (een plek op) deze pagina
